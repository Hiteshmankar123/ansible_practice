# ansible_practice 

## what is yaml ?
YAML Ain't Markup Language. It is DSL Language which can be used to store data in the form of key and value pairs.

## YAML Syntax:
key:value 
- YAML support indentation block 
- YAML start with '---' & end with '...'
- Data Types:-
    List 
    Dictionaries
    string
    Boolean


- list
---shell
-mango
-banana
-apple
...


- Dictionaries
---shell
Name:- Hitesh
Education:- BE
Emp_no:- 10223
...

- string (collection of words)
---shell
"hitesh"
"mankar"
"hello world"
...

- Boolean value
---shell
male:
